SMILE GAME - HOW TO RUN
=======================

WEB APP (Firebase login + game + MySQL user sync)
-------------------------------------------------
From the PROJECT FOLDER (contains package.json and web/):

  npm start

Then open: http://localhost:3000  (server listens on 0.0.0.0:3000 so localhost works)

This uses PHP’s built-in server on port 3000 so web/api/register.php runs and the
`smilegame.users` table can update. You must have `php` in your PATH (or use MAMP’s
PHP in the terminal).

If MySQL is not on port 3306 (e.g. MAMP → 8889), copy:
  web/api/db.local.example.php  →  web/api/db.local.php
and edit db.local.php with your port and password.

Static hosting only (Firebase game works, NO MySQL / PHP):
  npm run start:static

PHP on port 8080 if 3000 is already in use:
  npm run start:php
  → http://localhost:8080

Database setup: web/api/schema.sql

Java desktop: see root README.md
