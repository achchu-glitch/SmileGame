SMILE GAME - HOW TO RUN
=======================

1. Open a terminal in the PROJECT FOLDER (the folder that contains this "web" folder).

2. Run:
   npm start

3. Open your browser and go to the URL shown in the terminal, e.g.:
   http://localhost:3000

4. Sign in with email or Google, then play the game.

If you see 404: make sure you ran "npm start" from the folder that contains "web", and use the exact URL the terminal shows.
